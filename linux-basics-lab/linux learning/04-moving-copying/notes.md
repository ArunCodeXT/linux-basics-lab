# 04 — Copying & Moving

## Commands Learned
cp source dest → copy file
cp -r dir newdir → copy folders
mv old new → rename
mv file path/ → move file

## Examples Practiced
cp a.txt backup.txt
cp -r assets/ assets_copy/
mv notes.txt summary.txt
mv report.pdf Documents/

## Summary
I can move, copy, and reorganize files and folders with ease.
