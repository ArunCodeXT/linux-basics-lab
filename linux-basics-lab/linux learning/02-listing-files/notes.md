# 02 — Listing Files

## Commands Learned
ls → list files
ls -a → show hidden files
ls -l → detailed list
ls -lh → human-readable sizes
ls -R → recursive listing
ls -t → sort by time

## File Types (ls -l output)
- = regular file
d = directory
l = symbolic link

## Examples Practiced
ls -a
ls -l
ls -lh /etc
ls -R ~/Documents

## Summary
I can list, inspect, and understand files and directories with complete clarity.
