# 03 — File Management

## Commands Learned
touch file.txt → create file
echo "text" > file → write content
cat file → view file
mkdir folder → create folder
mkdir -p a/b/c → nested folders
rm file → delete file
rm -r folder → delete folder
rmdir emptyfolder → delete empty folder

## Examples Practiced
touch notes.md
mkdir projects
rm old.txt
mkdir -p backups/2024/january

## Summary
I can create, delete, and manage files and folders efficiently.
