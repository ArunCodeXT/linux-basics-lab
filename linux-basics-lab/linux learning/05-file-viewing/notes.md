# 05 — Viewing Files

## Commands Learned
cat → show full content
tac → reverse content
nl → numbered lines
more → page scroll
less → advanced scroll
head → first lines
tail → last lines
tail -f → live log view

## Examples Practiced
head /var/log/syslog
tail -f /var/log/auth.log
less notes.md

## Summary
I can view, scroll, and inspect files and logs confidently.
