# 01 — Navigation

## Commands Learned
- pwd → show current directory
- cd <folder> → move inside folder
- cd .. → go up one level
- cd ~ → go to home directory
- cd / → go to root directory
- cd - → switch to previous directory

## Filesystem Hierarchy
/ → root (starting point of everything)
/home → all user folders
/etc → configuration files
/var → logs and dynamic files
/bin, /usr/bin → executables
/tmp → temporary files

## Examples Practiced
pwd
cd Desktop
cd ..
cd /etc
cd -
cd ~/Documents

## Summary
I feel confident navigating anywhere in the Linux filesystem with precision.
